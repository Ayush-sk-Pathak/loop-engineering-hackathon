# Continuum

Premium B2B autonomous continuity engine. When a critical system boundary fails, Continuum discovers, verifies, pays, and procures backup resources without human intervention.

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Surfaces

| Route | Purpose |
|-------|---------|
| `/` | Brand landing + embedded live agent demo |
| `/console` | Immersive remediation theater |
| `/incidents` | Closed remediation history |
| `/learning` | Local resolution ledger (warm path) |
| `/suppliers` | Commercial registry |

Trigger the same scenario twice in the console to see warm-path acceleration via the learning layer (persisted in `localStorage`).
